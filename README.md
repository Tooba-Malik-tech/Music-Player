# Responsive Music Player

A sleek and responsive music player built using **HTML**, **CSS**, and **JavaScript**. It includes essential controls such as play, pause, next, previous, progress tracking, and volume adjustment, all wrapped in a clean, user-friendly interface that adapts well to desktop and mobile devices.

## Features

- Play, pause, next, and previous track controls
- Interactive progress bar to seek through the track
- Volume control slider
- Responsive design for all screen sizes
- Smooth animations and transitions
- Easily customizable to add your own music

## Demo

Open `index.html` in any modern web browser to try the player.

## Usage

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/responsive-music-player.git
````

2. Add your audio files to the `/music` folder or update the playlist array in `script.js`.
3. Open `index.html` in your browser.

## Technologies Used

* HTML5 (Audio element)
* CSS3 for styling and responsiveness
* Vanilla JavaScript for functionality and DOM manipulation
